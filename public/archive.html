Alert! Due to lack of funds, MapKnitter.org is offline. Read more »

Access an archived version from the Internet Archive

Before taking MapKnitter offline, we worked with the Internet Archive's Wayback Machine to store archived copies of every page on the site. That means your data will be preserved by the Internet Archive for... a long time.

Search for archived versions of this page

There are limitations: the pages are static, so no new changes can be saved. However, existing exports can still be downloaded.

We are working on a version of "MapKnitter Lite" that can be hosted for free, with image archiving based on the Internet Archive. If you would like to help make that happen, please reach out to Jeffrey Yoo Warren on Twitter: @jywarren
